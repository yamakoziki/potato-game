ネパール・ジャガイモ農家シミュレーションゲーム

ネパールの農村を舞台に、ジャガイモ栽培の一シーズンを体験するブラウザ型教育ゲーム。
栽培の各局面で判断を下し、5つのスコアを育てて最高ランクを目指す。

遊び方

index.html をブラウザで開くだけ。ビルド・サーバー不要。

ゲームの進行

1. ゲーム開始設定

設定項目 選択肢
作付け季節 夏作（モンスーン期）／冬作（乾季）
栽培地域 テライ平野 ／ 丘陵地帯 ／ 山岳地帯

2. 全体フロー

ゲーム開始
  → フェーズ1〜10（農作業の各局面）
      └ 各フェーズの合間にトラップイベント（季節依存）が発生することがある
  → 販売フェーズ（スコアに応じて選択肢が変わる）
  → 最終結果
      ├ 収穫スコア ≤ 5 → ゲームオーバー（農場崩壊）
      └ それ以外     → ランク判定

評価スコア（5項目）

すべて 0〜100 の範囲。初期値はすべて 50。

スコア名 意味
収穫量（harvest） 実際に収穫できたジャガイモの量
品質（quality） イモの商品価値・残留農薬・外観
健康（health） 農家・消費者の安全への配慮
環境（env） 土壌・水・生態系への影響
地域（community） 近隣農家や市場との関係

各フェーズの選択肢は、これらのスコアに 加算・減算 を与える（例：harvest: +10, quality: -5）。

フェーズ一覧と点数変化

フェーズ1｜品種・種芋選択 🌱

選択肢 主な点数変化
NARC認定種芋（クフリ・ジョティ）を購入 収穫+10、品質+10
昨年の自家採種を使う 収穫−5、品質−5
市場の安い種芋を購入 収穫−10、品質−15、健康−5

フェーズ2｜芋切・催芽処理 🔪

選択肢 主な点数変化
1片40〜60gに切り、草木灰で切り口処理 収穫+8、品質+5
大きいまま植える（切らない） 収穫+3、品質+3
大きく切りすぎて処理なしで植える 収穫−12、品質−8

フェーズ3｜土壌・畑の準備 🌍

選択肢 主な点数変化
土壌検査を実施してから作業 収穫+8、品質+10、環境+5
昨年と同じ場所に連作 収穫−15、品質−10、環境−5
深耕して別の畑で栽培（輪作） 収穫+10、品質+8、環境+8

フェーズ4｜施肥設計 🌿

選択肢 主な点数変化
牛糞堆肥＋適量の化学肥料（N:P:K＝100:60:80 kg/ha） 収穫+12、品質+10、環境+5、健康+5
化学肥料を大量投入 収穫−8、品質−12、環境−15、健康−10
牛糞堆肥のみ（化学肥料なし） 収穫−5、品質+5、環境+10、健康+8

フェーズ5｜畝立て・植付 🌾

選択肢 主な点数変化
高畝（30cm以上）＋適正株間（30×60cm） 収穫+10、品質+10
平畝で密植（株間を狭める） 収穫−10、品質−10、環境−5
等高線植付（山岳・丘陵地帯向け） 収穫+8、品質+8、環境+15

フェーズ6｜培土（土寄せ） ⛰️

選択肢 主な点数変化
株高20cmの時点で高めに培土 収穫+8、品質+12
培土のタイミングを遅らせる 収穫−5、品質−20、健康−15
2回培土（第1回＋追加）でしっかり高畝に 収穫+10、品質+15

フェーズ7｜灌漑管理 💧

選択肢 主な点数変化
土の乾燥具合を見て畝間灌漑（適量） 収穫+10、品質+8
雨頼みで灌漑なし 収穫−12、品質−8
たっぷり灌漑（過剰） 収穫−10、品質−12、環境−5

フェーズ8｜病害虫対応 🦠

選択肢 主な点数変化
症状確認後、適切な薬剤を散布（安全装備着用） 収穫+8、品質+8、健康+5
農薬を大量散布（過剰使用） 収穫+3、品質−5、健康−20、環境−15、地域−10
様子見（何もしない） 収穫−20、品質−15、地域−15

フェーズ9｜収穫判断 🥔

選択肢 主な点数変化
茎葉が枯れてから晴天日に収穫 収穫+12、品質+12
市場価格が高いうちに早めに収穫 収穫−5、品質−10、地域+5
雨の日でも急いで収穫 品質−18、環境−5

フェーズ10｜貯蔵・出荷 🏠

選択肢 主な点数変化
キュアリング処理後、適切な温度・湿度で貯蔵 収穫+5、品質+15、地域+10
収穫後すぐに全量出荷 品質+5、地域−5
光が当たる場所で野積み貯蔵 収穫−20、品質−20、健康−10

トラップイベント（突発的な災害）

フェーズの合間にランダムで発生。同じイベントは1回のみ。季節によって内容が異なる。

夏作のみ

突然の豪雨！モンスーン到来 🌧️（フェーズ7付近）

選択肢 主な点数変化
事前に排水溝を掘っておいた（準備済み） 収穫+5、環境+5
急いで排水溝を掘る 収穫−10、品質−8
何もできない（排水設備なし） 収穫−25、品質−20、環境−10

突然の雹（ひょう）！ 🌨️（フェーズ6付近）

選択肢 主な点数変化
わら被覆をしていた（事前準備済み） 品質+5
被害確認して追肥で回復を促す 収穫−8、品質−5
何もしない（様子見） 収穫−20、品質−15

冬作のみ

予想外の長期乾燥！ ☀️（フェーズ5付近）

選択肢 主な点数変化

わらマルチで水分蒸発を抑える 収穫−5、品質−3
何もしない（雨が降るまで待つ） 収穫−20、品質−15

サルの被害！（テライ地帯） 🐒（フェーズ8付近）

選択肢 主な点数変化
事前に設置した柵が機能した 収穫+5、地域+10
大声で追い払う 収穫−5、品質−5
何もできない 収穫−18、品質−10

販売フェーズ（フェーズ11）💰

スコアに応じて選択できる販売先が変わる。

販売先 解放条件 良好時の点数変化
市場（ハット）販売 🏪 常に選択可 地域+5
仲買人への販売 🤝 常に選択可 収穫+8、地域+5（収穫≥60の場合）
個人・直売販売 🛒 品質 ≥ 60 収穫+15、品質+5、地域+15
農協・契約栽培出荷 🤲 収穫 ≥ 55 かつ 品質 ≥ 55 収穫+10、品質+8、地域+20、環境+5

最終ランク

5つのスコアの 平均値 で判定。

平均スコア ランク
85以上 マスターファーマー
70〜84 ベテラン農家
55〜69 一人前の農家
40〜54 見習い農家
39以下 新米農家

収穫スコアが 5以下 になるとゲームオーバー（農場崩壊）。
